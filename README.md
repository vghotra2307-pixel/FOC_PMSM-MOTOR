This project implements a high-fidelity Field-Oriented Control (FOC) framework for a Permanent Magnet Synchronous Motor (PMSM) in MATLAB/Simulink. The core power stage is designed around a 6-switch Voltage Source Inverter (VSI) utilizing complementary MOSFET pairs to drive the three-phase stator windings. To maximize DC bus utilization and minimize current total harmonic distortion (THD), the switching patterns for the 6-switch topology are dynamically computed using a Space Vector Pulse Width Modulation (SVPWM) strategy.Motor torque and rotor dynamics are optimized through a nested closed-loop architecture:Outer Speed Control Loop: Regulates motor angular velocity via an anti-windup PI controller to generate the optimal quadrature-axis reference current ($i_q^*$).Inner Current Control Loop: Processes decoupled $i_d$ and $i_q$ error channels using dual synchronous PI controllers to precisely command the target voltage vectors.The feedback path features an optimized two-input Clarke transform configuration to eliminate zero-sequence noise and structured unit-delay blocks to resolve mathematical algebraic loop bottlenecks, ensuring real-world microcontroller simulation stability.
